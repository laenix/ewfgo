# EWF Parser 开发计划

## 项目目标
实现一个 Go 语言的 EWF (EnCase Evidence File) 解析器，支持：
- 读取 E01/S01 格式的取证镜像文件
- 解析分区表 (MBR/GPT)
- 支持常见文件系统目录列表 (FAT32, NTFS, ext4 等)

---

## 当前状态 (2026-03-11)

### ✅ 已完成
- E01 文件读取和解压
- MBR 分区表解析
- FAT32 根目录列表
- FAT32 子目录浏览 (完整支持多级目录)
- NTFS 真实目录列表 (160+ 文件)

### ⚠️ 待完善
- ext4 目录列表
- GPT 分区解析
- LVM 支持

## 技术架构

```
/mnt/d/code/ewfgo/
├── cmd/main.go           # CLI 入口
├── open.go              # 对外 API
├── open_files.go        # 文件系统 API
│
└── internal/
    ├── ewf.go           # EWF 核心解析
    ├── mbr.go           # MBR 解析
    ├── gpt.go           # GPT 解析
    │
    └── filesystem/
        ├── fs.go        # 接口定义
        ├── fat32.go     # FAT32 实现 ✅
        ├── ntfs.go      # NTFS 实现 ✅
        ├── ext4.go      # ext4 实现 ⚠️
        └── lvm.go       # LVM 实现 ❌
```
